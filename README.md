# Scommunity

### 1. 개발환경

- Open JDK 17
- JavaScript
- IntelliJ IDEA
- MySQL Server8.0.29
- SpringBoot version 3.2.2

### 2. 라이브러리, 프레임워크

- React.js
- lombok
- maven
- OAuth, Security

### 3. Explanation

- 서울 지역 커뮤니티 게시판 CRUD
- 회원가입(소셜로그인, 이메일)
- RESTApi 구조로 설계
- 중복가입 방지
- config 셋팅(DB Connection)
